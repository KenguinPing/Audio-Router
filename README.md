# 音频一键切换

一个免安装的 Windows 小工具，把一组“输出设备 + 输入设备”保存为方案，之后一键同时切换：

- Windows 默认设备
- Windows 默认媒体设备
- Windows 默认通信设备
- Discord 语音（设置为 `Default / 默认` 时）
- Steam 语音（设置为 `Default / 默认` 时）

## 使用

1. 双击 **`启动音频切换器.vbs`**。
2. 选择输出设备和输入设备，输入方案名（例如“桌面音箱”“无线耳机”），点“保存”。
3. 以后点方案右侧的“一键切换”即可。关闭或最小化主窗口后，程序会留在系统托盘；双击托盘图标可恢复窗口，右键也能快速切换。

托盘右键菜单会自动跟随 Windows 的浅色 / 深色应用模式，并显示当前正在使用的音频方案。菜单中也可以刷新设备、切换开机启动或真正退出程序。

主窗口“我的方案”右侧提供两个选项：

- **随 Windows 启动**：写入当前用户的启动项，开机后自动进入托盘，不需要管理员权限。
- **启动时隐藏到托盘**：平时双击启动器时也不显示主窗口，直接进入托盘。

第一次使用时，请分别打开：

- Discord → 用户设置 → 语音和视频 → 输入设备、输出设备均选 `Default / 默认`
- Steam → 好友与聊天 → 设置 → 语音 → 语音输入设备、语音输出设备均选 `Default / 默认`

方案保存在 `%LOCALAPPDATA%\AudioSwitch\profiles.json`，删除程序文件不会误删方案。

## 系统要求

- Windows 10 或 Windows 11
- Windows PowerShell 5.1（系统自带）

程序不需要管理员权限，也不会联网或修改 Discord、Steam 的私有配置文件。

开发排查时可运行 `powershell -NoProfile -ExecutionPolicy Bypass -File .\AudioSwitch.ps1 -SelfTest`，仅检测音频接口和设备，不会切换设备或打开窗口。
